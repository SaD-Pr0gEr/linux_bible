VIM/VI
    стрелки up/down/right/left - чтоб перенести курсор в соответствующие стороны
    h/l/j/k - также перенос курсора влево/вправо/вниз/вверх
    w - перенос курсора в начало след. слова с пробелами, запятыми и отступами
    W - такой же как w, но запятые не считаются
    b - перенос курсора в начало пред. слова с пробелами, запятыми и отступами
    B - такой же как b, но запятые не считаются
    0 - перенос курсора в начало текущей строки
    $ - перенос курсора в конец текущей строки
    H - перенос курсора на верхний левый угол экрана
    M - перенос курсора на первый символ средней строки на экране
    L - перенос курсора в нижний левый угол экрана(последняя строка на экране)

    x — удаляет символ под курсором
    X — удаляет символ непосредственно перед курсором
    d<?> — удаляет часть текста
    c<?> — изменяет часть текста
    y<?> — копирует часть текста
    dw — удаляет (d) слово (w) после курсора
    db — удаляет (d) слово (b) перед курсором
    dd — удаляет (d) всю текущую строку (d)
    c$ — изменяет (c) символы (фактически стирает их) от текущего символа до
        конца строки ($) и переходит в режим ввода
    c0 — изменяет (c) (опять же стирает) символы с предыдущего символа до начала
        текущей строки (0) и переходит в режим ввода
    cl — стирает (c) текущую букву (l) и переходит в режим ввода
    cc — стирает (c) строку (c) и переходит в режим ввода
    yy — копирует (y) текущую строку (y) в буфер
    y) — копирует (y) текущее предложение ()) справа от курсора в буфер
    y} — копирует (y) текущий абзац (}) справа от курсора в буфер

    3dd — удаляет (d) три (3) строки (d), начинающиеся с текущей строки
    3dw — удаляет (d) следующие три (3) слова (w)
    5cl — изменяет (c) следующие пять (5) букв (l), то есть удаляет буквы и переходит в режим ввода
    12j — перемещается вниз (j) на 12 строк (12)
    5cw — стирает (c) следующие пять (5) слов (w) и переходит в режим ввода
    4y) — копирует (y) следующие четыре (4) предложения ())

    P — помещает скопированный текст слева от курсора, если текст состоит из букв
        или слов, и над текущей строкой, если содержит строки.
    p — помещает текст справа от курсора, если он состоит из букв или слов, и ниже
        текущей строки, если содержит строки.
    . - Повтор последней команды
    u - отменить предыдущее изменение
    Ctrl+R - Если вы отменили лишнее, используйте сочетание клавиш Ctrl+R, чтобы
        вернуть отмененное
    Ctrl+g. Если вы забыли, что именно редактируете, то нажмите эти клавиши —
        и в нижней части экрана отобразятся имя редактируемого файла и строка, на
        которой вы находитесь.

    Ctrl+f — прокрутка вперед по одной странице за раз
    Ctrl+b — прокрутка назад по одной странице за раз
    Ctrl+d — прокрутка вперед на половину страницы за раз
    Ctrl+u — прокрутка назад на половину страницы за раз
    G — переход к последней строке файла
    <n>G — переход к <n> строке файла

    /hello — поиск впереди слова hello
    ?goodbye — поиск позади слова goodbye
    /The.*foot — ищет впереди строку, в которой есть слово The, а после него — слово foot
    ?[pP]rint — ищет позади слова print и Print

    :g/Local — ищет в файле слово Local и выводит каждое его появление. (Если
        данных больше, чем умещается на экран, они передаются по конвейеру в команду more)
    :s/Local/Remote — заменяет Remote на первое появление слова Local в текущей строке
    :g/Local/s//Remote — заменяет первое появление слова Local в каждой строке файла словом Remote
    :g/Local/s//Remote/g — во всем файле заменяет каждое появление слова Local словом Remote
    :g/Local/s//Remote/gp — во всем файле заменяет каждое появление слова Local
        словом Remote, а затем выводит каждую строку с изменениями

locate
    locate filename - ищет filename везде где возможно(в зависимости от прав пользователя)
    locate -i filename - делает то же что выше, но ищет filename без учёта регистра

find
    find /etc
    find $HOME -ls - параметр ls как аналог (ls -l)

    # Search by name
    find /etc -name passwd - параметр name ищет по имени файла/папки
    find /etc -iname '*passwd*' - параметр iname работает также как name но в кавычках можно юзать regex паттерны * и ?

    # Search by size
    find /usr/share/ -size +10M       ------->
    find /usr/share/ -size 10M         ---------> параметр size ищет по размеру файла (- меньше, + больше, 10M равно 10МБ)
    find /mostlybig -size +5M -size -20M ---->

    # Search by user
    find /home -user chris -ls - ищет файлы/папки по имени пользователя

    # Search by user group
    find / -group root -type f -ls - ищет файлы которые принадлежат группе root

    # Search by permissions
    find /usr/bin -perm 755 -ls  ------------>
    find /myreadonly -perm /222 -type f      ----> ищет файлы/папки по правам пользователя
    find /home/chris/ -perm -222 -ls -------->

    # Search by file type
    find /opt -type f - параметр -type ищет по типу файла(f, d, l)

    # Search by time
    find dir -amin -5 -type f -> находит все файлы в каталоге dir, которые были прочитаны до 5 минут назад
    find dir -amin +5 -type f -> находит все файлы в каталоге dir, которые были прочитаны от 5 минут назад
    find -name test2 -amin +5 -amin -10 -type f -> находит все файлы в каталоге dir, которые были прочитаны от 5 минут до 10 минут назад

    # and/or(-o)/not
    find /var/allusers \( -user joe -o -user chris \) -ls -> Находит файлы, принадлежащие пользователям joe и chris
    find /var/allusers/ -user joe -not -group joe -ls -> Находит файлы, принадлежащих пользователю joe, но только тех, которые не назначены группе joe:
    find /var/allusers/ -user joe -and -size +1M -ls -> Находит файлы, которые принадлежат пользователю joe и более 1 Мбайт

    # exec/ok
    find [ параметры ] -exec [ команда ] {} \;
    find [ параметры ] -ok [ команда ] {} \;
    find /etc -iname passwd -exec echo "I found {}" \;
    find /usr/share -size +5M -exec du {} \; | sort -nr
    find /var/allusers/ -user joe -ok mv {} /tmp/joe/ \;

grep
    grep desktop /etc/services
    grep -i desktop /etc/services -> Ищет desktop без учёта регистра
    grep -vi tcp /etc/services -> Ищет всё что кроме tcp
    grep -rli peerdns /usr/share/doc/ -> Ищет рекурсивно слово peerdns во всех файлах внутри папки /usr/share/doc/
    ip addr show | grep inet -> Ищет inet в выходных данных от ip addr show
